# WE'RE ALL RELATED Feud

*WE'RE ALL RELATED Feud* is a remake attempt of the daytime game show, "Family Feud", hosted by Steve Harvey.


## Technologies Used:

- React
- HTML
- CSS
- Node.js
- Express
- MongoDB & MongoDB Atlas
- Mongoose


## Getting Started
